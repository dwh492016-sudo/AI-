# AI- 陪伴式AI日记软件

## 📖 项目介绍
这是一款能真正“记住你”的本地AI日记/陪伴小助手。你可以和AI畅聊日常，AI会根据聊天内容本地记录，还能用向量检索调取历史内容，做到不会遗忘你们的故事。适合需要陪伴、有长期记忆需求、隐私在意的朋友。

---

## 🚩 模式说明与模型推荐

### 1. 【混合模式】(推荐普通用户)
- **本地记录与向量检索**：千问7B（小巧快速）
- **聊天模型**：DeepSeek（API，需自备Key，可能产生费用）
- **优势**：配置要求低，聊天体验佳，功能完整
- **隐私**：本地记忆私有，聊天内容根据API隐私政策传输

### 2. 【完全本地模式】(推荐高性能用户)
- **本地记录与向量/聊天模型**：千问7B+千问14B
- **需求**：建议有GPU，32GB内存以上
- **优势**：一切数据/聊天都在本地，100%隐私
- **适合**：注重隐私，有高硬件支持者

### 3. 【完全API模式】(实验/高级用户)
- 使用自己定义的API（如GPT等），作者未充分测试
- 配置与隐私请自行评估

---

## 💡 模型选择说明

- 千问7B、14B用于本地记忆与轻量聊天
- DeepSeek 聊天更贴合自定义角色体验，长期对话不容易“跑题”，适合角色扮演
- 如果硬件条件很好，推荐用千问14B做全流程，体验本地高质量对话

## 🔥 已有/规划功能

### ✅ 已实现
- 长期记忆（向量数据库本地检索）
- 角色自定义
- 人物档案（自动生成）：系统会根据与你的聊天内容中说到的人物自动生成“人物档案”。如果在日记里提到“我有个朋友叫XXX”，系统会把相关日记自动打上这个人物的标签，形成归档，方便回忆。当前功能自动化识别和关联仍有限，需明确表述人物，功能后续会持续优化。
- 记忆碎片（记忆片段）功能
- 部分语音交互

### 🔄 规划/开发中
- 桌宠（在桌面养一个属于自己的AI形象）
- 智能识别&弹窗确认式闹钟（可“和AI聊天”里自然设定闹钟）
- 倒数日/待办事项（写下计划，到期AI主动提醒）
- 更友好的语音体验
- 移动端（手机版）计划中
- 更多实用小功能

> 说明：有些功能因作者token额度用完暂缓开发，后续有额度或开发者协助会持续推进

## 🔐 隐私与数据安全

- **本地数据始终隐私**  
- **混合/完全API模式**：与API模型交互的数据会被发送到对应厂商服务器，请用户自行决定是否接受  
- **完全本地模式**则所有数据封闭在本地，绝无泄漏
