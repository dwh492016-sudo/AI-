# AI- 陪伴式AI日记软件

## 📖 项目介绍
可以和AI分享你的日常，AI会根据聊天内容记录，还会向量调取记录过的事情，回答你，做到真正记住你，不忘记你们聊过的事情。

2. 双击运行 `AI-.exe` 即可启动
3. 首次使用请按照提示完成初始化配置

## 🛠️ 技术说明
- 基于本地大模型+向量数据库实现长期记忆
- 支持自定义AI角色、语音交互（后续更新）
- 完全本地运行，保护隐私
